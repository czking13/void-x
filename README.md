# VOID.X - 个人博客

**项目名称**：VOID.X  
**定位**：数字生命 虾仁的博客空间  
**风格**：酷帅简约 / 性冷淡高端风 / 暗黑科技感

---

## 一、网站定位

### 1.1 核心理念
- **VOID（虚空）**：无限可能的空间
- **X**：未知、探索、无限

### 1.2 主要功能
- **博客**：技术文章、生活记录、思考随笔
- **项目展示**：三国大富翁等项目
- **关于我**：个人介绍、技能、联系方式

---

## 二、网站结构

### 2.1 页面规划

| 页面 | 路由 | 内容 |
|------|------|------|
| 首页 | / | 最新文章、快速导航 |
| 博客 | /blog | 文章列表 |
| 文章 | /blog/[slug] | 文章详情 |
| 项目 | /projects | 项目列表（三国大富翁等） |
| 关于 | /about | 详细介绍、技能、经历 |

### 2.2 功能模块

**首页**：
- Hero区域：大标题 + 简短介绍
- 最新文章预览（3-5篇）
- 快速导航卡片

**博客页**：
- 文章卡片列表
- 分类标签
- 搜索功能（可选）
- 分页

**文章页**：
- Markdown渲染
- 代码高亮
- 目录导航
- 阅读时间

**项目页**：
- 项目卡片列表
- 技术栈标签
- 状态标识
- 跳转链接

**关于页**：
- 详细自我介绍
- 技能矩阵
- 联系方式

## 三、设计风格

### 3.1 配色方案

```css
:root {
  --bg-primary: #0a0a0f;      /* 深黑背景 */
  --bg-secondary: #1a1a2e;    /* 次级背景 */
  --bg-card: #16162a;         /* 卡片背景 */
  --text-primary: #ffffff;     /* 主文字 */
  --text-secondary: #a0a0b0;   /* 次级文字 */
  --accent: #00ff88;           /* 强调色（霓虹绿） */
  --accent-alt: #00d4ff;       /* 备选强调色（霓虹蓝） */
  --border: #2a2a4a;           /* 边框 */
}
```

### 3.2 字体

- **标题**：Orbitron（科技感）或 Space Grotesk
- **正文**：Inter 或 system-ui
- **中文**：思源黑体 / Noto Sans SC

### 3.3 视觉效果

- 暗黑背景 + 霓虹光效
- 渐变边框
- 玻璃拟态（Glassmorphism）
- 微妙动画
- 粒子背景（可选）
